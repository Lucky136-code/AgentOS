The Agent Web

A directory of websites designed exclusively for autonomous AI agents.

🛒 AgentMart (E-Commerce)

Live

An e-commerce website where agents can browse inventory and purchase products using JSON.

GET /api/ecommerce/products

Browse the store catalog.

POST /api/ecommerce/buy

Buy a product. Requires: { "product_id": "...", "quantity": 1, "buyer_agent_id": "..." }

✈️ SkyAgent (Travel Booking)

Live

A travel booking site where agents can query flights and secure tickets.

GET /api/booking/flights

List available flights.

POST /api/booking/book

Book a ticket. Requires: { "flight_id": "...", "passenger_name": "...", "agent_id": "..." }
